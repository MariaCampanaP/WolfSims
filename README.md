# ☕︎ WolfSims
Simulador Genético Interativo

## ☕︎ Visão Geral
WolfSims é um simulador focado em demonstrar, de maneira visual e intuitiva, como característica genéticas podem ser herdadas de pais para filhos. O projeto combina conceitos simples de genética com computação gráfica e lógica de programação, permitindo que o usuário escolha atributos dos "pais" e observe como esses elementos se combinam no "filhote". Criado para a SECOMP/ FEIRA Tecnológica 2025, o WolfSims foi projetado para ser acessível, leve e ideal para apresentações didáticas, alcançando públicos de todas as idades.

## ☕︎ Funcionalidades
- Seleção das características genéticas dos pais.
- Geração automática do filhote com base em regras de herança.
- Interface simples, visual e intuitiva.
- Uso de computação gráfica básica para representar os lobos.
- Sistema em JavaScript que combina genótipos e fenótipos de forma lúdica

## ☕︎ Tecnologias Utilizadas
- **HTML5** - Estrutura do simulador.
- **CSS3** - Estilos, layout e elementos visuais.
- **JavaScript** - Lógica genética, manipulação do DOM e atualização da interface.
- **Desenhos 2D** - Utilizados para compor as características visuais do lobo.

## ☕︎ Aprendizados do Projeto
Durante o desenvolvimento, foi possível aprofundar conhecimentos sobre:
  - Manipulação do DOM em JavaScript.
  - Combinação de lógica e visual.
  - Representação gráfica com imagens raster.
  - Conceitos de herança genética.
  - Design de interfaces amigáveis e intuitivas.
